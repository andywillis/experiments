The Venda Solr JSON config interface.

Click one of the buttons above to choose which area of the JSON you wish to amend.

QUERY OPTIONS - only available for those in the Venda Group.

HISTORY - there is a history feature located at the top of the code window that will allow you to go back and forth through the changes you've made.

SAVE - at any time.

Root

Root contains all those properties in the root of configuration tree that are themselves not containers for other properties.

Query options

Weightings

Show available types from the dropdown list, or enter a type prefixed with 'pdxt'.

Fields

Show available array types from the dropdown list, or enter a type prefixed with 'pdxt'.

Ranges

Choose an available type and then enter a range into the value input box in the following format: 

[number]...[number]
where both numbers are whole integers, and the first number is greater than zero and less than the second number. There is currently no limit to the size of the second number.
